Lenguaje de programación C++


C++ es un lenguaje de programación orientado a objetos y además es una evolución a partir del idioma C. Aunque no recomendaría este lenguaje a un principiante, sí que es cierto que es bastante esencial y se utiliza en diversas áreas: en distintos software, navegadores web, sistemas de distribución, bases de datos complejas o videojuegos más avanzados. Por ejemplo, se ha utilizado el lenguaje C++ para el paquete de programas de Adobe, Google Chrome y Mozilla Firefox, y el sistema operativo Windows.
